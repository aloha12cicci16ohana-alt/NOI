# /claw - NOI 戦略・運営ドキュメント

このフォルダは、NOI事業に関する戦略分析・提案・議事録・TODOを管理する場所です。
Claw（AI）との相談内容を蓄積し、随時更新していきます。

## ファイル構成

| ファイル | 内容 |
|---------|------|
| `strategy.md` | 事業戦略の全体分析・提案（メインドキュメント） |
| `todo.md` | やるべきことリスト（優先順位つき） |
| `hp-fixes.md` | HP修正・ブラッシュアップの詳細タスク |
| `changelog.md` | 相談・修正の履歴ログ |

## 運用ルール

- 相談するたびに `changelog.md` に日付・内容を追記
- 方針変更があれば `strategy.md` を直接更新（古い内容は changelog に記録）
- 実行したタスクは `todo.md` で ✅ に変更
