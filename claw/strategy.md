## 1. 事業全体の評価

### 3事業の構造

| 事業 | 成熟度 | 売上予測 | 戦略的位置 |
|------|--------|----------|------------|
| **Web / AI / App** | ★★★ すぐ売れる | 即時キャッシュ | キャッシュエンジン |
| **Wedding Produce** | ★★ 仕込みが必要 | 中期収益 | ブランドの核・差別化の源泉 |
| **Wedding SaaS / App** | ★ これから | 長期収益 | スケーラブルな将来資産 |

**評価:**
「今日の飯＝Web/AI」「明日のブランド＝Wedding」「未来の資産＝SaaS」と時間軸が分散しており、個人事業としては非常に賢い設計。ただし同時に3つ走らせると全部中途半端になるリスクがある。

## 2. 勝ち筋

### 核心：「越境」にある

NOIの最大の武器は、**ウェディングプロデューサー × Web/AIクリエイター × ブランディング設計者**という組み合わせが市場にほぼ存在しないこと。

| 競合タイプ | できること | できないこと |
|-----------|-----------|-------------|
| フリーランスWeb制作者 | HP制作 | 事業戦略・ブランド構造の設計 |
| Web制作会社 | HP + 運用 | 感性的なブランド設計・少人数婚の理解 |
| ウェディングプランナー | 式のプロデュース | デジタル制作・AI活用 |
| Wedding SaaS企業 | ツール提供 | 現場の「肌感」と感性的デザイン |

> **「人生の節目（Wedding）と事業の節目（Webブランディング）の両方を"伝わる形"にできるクリエイティブパートナーは他にいない」**

## 3. 収益化の優先順位

```
Phase 1（0-3ヶ月）：Web/AI で月30-50万を安定させる ← 今ここ
Phase 2（3-6ヶ月）：Wedding Produceの仕込み＋初受注
Phase 3（6-12ヶ月）：Wedding本格稼働 + SaaS企画着手
```

**理由:**
- Weddingは「信頼」で受注。信頼構築には3-6ヶ月かかる
- Web/AIは「実績＋HP」で今すぐ受注可能
- キャッシュフローがないとWeddingの仕込み期間を耐えられない
- Web/AI案件でWedding関連事業者（式場・花屋・フォトグラファー）と繋がれば、Weddingへの導線も作れる

## 6. 集客戦略

### Web/AI事業

```
Instagram（作品・Before/After）
  → プロフィールリンク → NOI HP
  → Works → Service → Contact
  → 無料ヒアリング（30分）
  → 提案書 → 受注
```

**Instagram運用:**
- 投稿テーマ: ①制作実績Before/After ②デザインTips ③クライアントの声 ④制作プロセス
- 投稿頻度: 週3-4本（リール1 + カルーセル2-3）
- ハッシュタグ: `#フリーランスHP制作` `#個人事業主ブランディング` `#Webデザイナーと繋がりたい`

### Wedding事業

```
Instagram（世界観・実例写真）
  → ストーリーズで「大人の余白婚とは？」教育
  → NOI Wedding専用LP
  → LINE公式 or Contact → 初回無料相談
  → 対面ヒアリング → 受注
```

> **重要: WeddingとWeb/AIのInstagramアカウントは分ける**

| | Web/AI | Wedding |
|--|--------|---------|
| トーン | シャープ、プロフェッショナル | 静か、温かい、余白 |
| 色味 | ライムグリーン＋ダーク | アイボリー＋ベージュ＋ゴールド |
| フォロワー像 | 事業主・クリエイター | 30代以上の女性 |
